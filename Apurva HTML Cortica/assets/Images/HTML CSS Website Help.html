To connect a JavaScript (.js) file to your HTML screen, follow these steps:

1. Create a JavaScript File

Save your JavaScript code in a file, e.g., script.js.

2. Link the JavaScript File in HTML

Place the following <script> tag before the closing </body> tag in your HTML file:

html
<script src="script.js"></script>

Example Structure:

index.html

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript Connection</title>
    <link rel="stylesheet" href="style.css"> <!-- Linking CSS -->
</head>
<body>

    <h1>Welcome to My Website</h1>
    <button onclick="showMessage()">Click Me</button>

    <script src="script.js"></script> <!-- Linking JavaScript -->
</body>
</html>

script.js

js
function showMessage() {
    alert("Hello! JavaScript is connected.");
}

Key Points:

  1. Place the <script> tag before </body> to ensure the HTML loads before JavaScript executes.
  2. Ensure the src="script.js" path is correct (it should match the actual location of your .js file).
  3. You can also use the defer attribute in the <head> if you want to keep it there:
    html
    <script src="script.js" defer></script>
    • This tells the browser to load the JavaScript file after the HTML content is loaded.

Now, your JavaScript file is connected, and the showMessage() function will run when the button is clicked! 🚀 Let me know if you need more help! 😊

3️⃣ JavaScript: Handle Form Submission (script.js)

js
document.getElementById("registrationForm").addEventListener("submit", function(event) {
    event.preventDefault(); // Prevent page reload

    // Get form values
    let email = document.getElementById("email").value;
    let name = document.getElementById("name").value;
    let course = document.getElementById("course").value;
    let mobile = document.getElementById("mobile").value;
    let comments = document.getElementById("comments").value;

    // Get selected payment modes
    let paymentModes = [];
    document.querySelectorAll('input[name="payment"]:checked').forEach((checkbox) => {
        paymentModes.push(checkbox.value);
    });

    // Save Data (For now, we log it in the console)
    let formData = {
        email: email,
        name: name,
        course: course,
        mobile: mobile,
        paymentModes: paymentModes,
        comments: comments
    };

    console.log("Form Data Saved:", formData);

    // Show success message
    document.getElementById("messageBox").style.display = "block";

    // Clear form after submission
    document.getElementById("registrationForm").reset();
});

✅ How It Works

  1. User enters details:
    • Email
    • Name
    • Course (Dropdown)
    • Mobile Number (10 digits)
    • Payment Mode (Checkboxes: Cash, Cheque, UPI, NEFT)
    • Comments
  2. Click "Send Message" → Data is saved and shown in the console.
  3. A success message appears.
  4. The form resets after submission.
